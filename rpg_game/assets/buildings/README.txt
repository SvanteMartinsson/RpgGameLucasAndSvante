SVANTRENISH — byggnads-facings (52 = 13 typer x 4 vyer)
========================================================
Namngivna som originalen + vy-suffix:
  {namn}_front  = c1  FRONT   (dörr mot kameran / söder)
  {namn}_q1     = c2  ¾-vy    (sido-ingång; skylt-sidan = ingången)
  {namn}_back   = c3  BAKSIDA  (ingen synlig dörr; ingång norrut, cobble-ledd)
  {namn}_q2     = c4  ¾-vy    (andra sidan)

13 typer: apothecary, barracks, blacksmith, church, cottage, gatehouse,
inn, shop, shrine, stable, tower, town_hall, warehouse.

justerbar konstant vid LOAD-TID i renderaren — inte inbränt.
